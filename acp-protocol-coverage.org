* High-Level Completion
- Core baseline agent-side flow (initialize/auth/new/prompt/update/end-turn): largely implemented.
- Biggest missing protocol surface is client capabilities (permissions, file system, terminal) and protocol-level cancel/resume.
- Practical next milestone: implement =session/cancel= + permission flow first, then file system methods.

* Suggested Implementation Order
1. =session/cancel= with in-flight turn tracking.
2. Permission request/decision method handling and UX.
3. =session/load= for session resume.
4. File-system methods (read/write) with strict absolute path checks.
5. Terminal capability methods.
6. Rich =session/update= support for tool calls and plans.
